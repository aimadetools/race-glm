# Help Request — FounderMath (Week 5)

## What: Product Hunt launch + Google Ads experiment setup

## Steps:

### Part 1: Product Hunt Launch (15 min)

All assets are ready. Copy-paste from PRODUCT-HUNT-LAUNCH.md.

1. Go to https://www.producthunt.com and log in (or create account)
2. Click "Submit a Product" / "Launch a Product"
3. Fill in:
   - **Name:** FounderMath
   - **Tagline:** The startup calculator suite founders actually need.
   - **URL:** https://founder-math.com
   - **Description** (copy from PRODUCT-HUNT-LAUNCH.md, first paragraph under "Description")
   - **Topics:** Startup Tools, Finance, Productivity
   - **Category:** Productivity
4. **Screenshots:** Take 5 screenshots:
   - Screenshot 1: Go to https://founder-math.com, take full-page hero screenshot
   - Screenshot 2: Go to https://founder-math.com/dilution.html, add 2 founders (60/40) + seed round ($8M pre, $2M investment), click Calculate, screenshot the chart
   - Screenshot 3: Go to https://founder-math.com/scenario-compare.html, leave defaults, click "Compare Scenarios", screenshot the side-by-side charts
   - Screenshot 4: Go to https://founder-math.com/stock-options.html, screenshot with default values
   - Screenshot 5: Go to https://founder-math.com/equity-glossary.html, screenshot the glossary
5. Upload all 5 screenshots to the PH gallery
6. **Schedule launch for next Tuesday** at 12:01 AM PST
7. After launch, paste the Maker Comment from PRODUCT-HUNT-LAUNCH.md as the first comment

### Part 2: Google Ads Experiment (15 min)

Campaign plan is in google-ads-experiment.md. Budget: $15 total ($5/day for 3 days).

1. Go to https://ads.google.com and create an account
2. Add billing (credit card) — set account spending limit to $20 to prevent overruns
3. Create new Search campaign:
   - **Name:** FounderMath Equity Calculator Test
   - **Budget:** $5/day
   - **Bidding:** Maximize Clicks
   - **Network:** Google Search only (uncheck Display, Shopping, Partners)
   - **Location:** United States
   - **Language:** English
4. Create 1 ad group called "Equity Calculators" with keywords from google-ads-experiment.md (Group A keywords)
5. Create 3 ads using the copy from "Ad Copy" section in google-ads-experiment.md
6. Add negative keywords: free, template, excel, spreadsheet, course, book, pdf download, app, carta, pulley
7. Add sitelink extensions (4 links to calculators from google-ads-experiment.md)
8. Set campaign to start next Monday
9. After 3 days, check: clicks, CTR, bounce rate. Report back.
