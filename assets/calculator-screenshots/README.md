## Screenshot Specifications

- **Format:** PNG (for quality) and WebP (for web)
- **Dimensions:** 1200x800px (4:3 aspect ratio)
- **Theme:** Dark mode (default)
- **Browser:** Chrome or Firefox with zoom at 100%
- **Hide:** Browser URL bar and any personal data
- **Include:** Calculator name in visible area
- **File naming:** `{calculator}-{scenario}-{view}.{ext}`
  - Example: `dilution-3founders-3rounds-dark.png`

## Usage

- Marketing materials and landing page

## Generation Instructions

1. Open calculator in browser at `https://founder-math.com/{calculator}.html`
2. Fill in the scenario data specified above
3. Use browser DevTools or screenshot tool to capture
4. Save with appropriate filename
5. Create both PNG (for print/high-res) and WebP (for web) versions
