## Usage Tips

1. **Hacker News "Show HN":** Use the Dilution demo URL as the primary example
2. **Twitter/X:** Shorter URLs with descriptive anchor text work best
3. **Reddit:** Include context with the URL (not just the link)
4. **Blog posts:** Embed the demo URL in relevant callout boxes

---

## Creating Custom Demo URLs

1. Open a calculator
2. Set up your desired scenario
3. Click "Share" button
4. Copy the URL
5. Test the URL in incognito window
